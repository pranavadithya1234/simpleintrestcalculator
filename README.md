Simple Interest Calculator
A minimal and intuitive React application to calculate simple interest quickly and efficiently. Built as a practical demonstration of React fundamentals, state management, and modern component-based architecture.

🚀 Features
Instant Calculation: Input principal amount, interest rate, and time to get real-time interest calculation.
Interactive UI: Smooth and responsive user interface built using modern React.
Component-Driven: Clean, modular codebase showcasing best practices in React frontend development.
Cross-Browser Compatible: Works seamlessly across modern browsers (Chrome, Firefox, Safari).
🛠️ Technology Stack
Frontend Framework: React 19 (Hooks, Functional Components)
DOM Rendering: React DOM
Testing: Jest, React Testing Library
Build Tool: Create React App (react-scripts)
